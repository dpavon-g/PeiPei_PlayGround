▼Javascript

Javascript es un lenguaje de programación que se utiliza para crear páginas web interactivas. Ahora veremos el contenido que entra en el examen:

Bucles

Existen dos tipos de bucles, los bucles for y los bucles while.

Condicionales

Existen dos tipos de condicionales, los condicionales if y los condicionales switch.

Inputs (Select, Radio, Checkbox, TextArea)

Existen varios tipos de inputs, los inputs select, radio, checkbox y textarea.

Formularios (Validaciones y Expresiones Regulares)

Los formularios son una parte importante de las páginas web, por lo que es importante saber cómo validarlos. Para validarlos se utilizan expresiones regulares.

Arrays

Los arrays son una estructura de datos que nos permite almacenar varios valores en una sola variable.
